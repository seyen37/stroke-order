### 午段：808 字實證分析

**4 份 PDF 偵察**（PUB_*）
- PUB_1570754349.pdf（2016, 34 頁）→ 808 表官方 Booklet（TCS 出版）
- PUB_1655945242.pdf + PUB_jp_1585037726.pdf（2019）→ TCVD 2nd ed 中日韓共用詞典
- PUB_1689037628.pdf（2023, 113 頁）→ 中日韓共用漢字集 Workbook（**含書法家手寫字帖** ← 未來 5d anchor reference 寶藏）
- 確認源頭：2014 NATF 第 8/9 屆會議協議

**程式 audit**
- 用 cjkvi-ids（88,937 entries, 2 MB）解析 IDS
- bug 找：parser regex `[︀-️0-F]` 含隱形 PUA 字符，吞掉所有 CJK
- 修：用 explicit Unicode range + `chr()` 構造避開 invisible PUA
- 結果差異：194 → **200 獨特組件**（compound markers 處理修正）

**核心實證結果**
- 808 字 → 200 獨特葉組件
- 平均每字 3.77 個葉組件
- **96.1%** 覆蓋率（3,363 / 3,500 常用字）
- **5.16x** 真寫:組合槓桿率
- 寫進 `docs/analysis/808_coverage_report.md` + `.json`
